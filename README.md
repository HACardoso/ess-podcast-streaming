### Dependências

- React
- Node
- npm

### Instalando dependências 
```
sudo apt install npm
```

### Modificando a versão do Node
```
sudo npm install -g n
n 14.15
```

### Iniciando o projeto
```
cd web-app
sudo npm install
cd ..
cd server
sudo npm install
```
### Rodando o projeto
Iniciar o servidor
```
cd server
npm start
```
Iniciando o frontend
```
cd web-app
npm start
```
